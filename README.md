# Fleurs-de-Lea site refresh

A lightweight multi-page static website focused on local SEO and conversion improvements.

## Implemented improvements
- Unique page titles and meta descriptions
- Local landing pages for Fairfax, Alexandria, Arlington, and Falls Church
- `Florist` schema markup on the homepage
- Click-to-call header and responsive CTA buttons
- Pricing ranges for bouquets, weddings, and classes
- Dedicated floral design classes page
- Booking/inquiry forms with occasion, budget, and date inputs
- `robots.txt` and `sitemap.xml`
- Descriptive image alt text throughout
- Journal page for future SEO content marketing

## Run locally
```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Before launch
1. Add the exact studio street address and business hours to the schema.
2. Connect forms to Formspree, Netlify Forms, or a CRM.
3. Submit `sitemap.xml` in Google Search Console for `https://fleurs-de-lea.com`.
4. Update the Google Business Profile with matching NAP details and fresh photos.
5. Swap in optimized WebP photography from the live brand library if desired.
